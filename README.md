# @xuanmo/prettier-config

我的 Prettier 配置

## 安装

```bash
npm i @xuanmo/prettier-config -D
```

## 使用

在 `package.json` 追加
```text
// config
"prettier": "@xuanmo/prettier-config"

// scripts
"format": "prettier --cache --write ."
```
